# Mini RTOS for STM32F407

Learning project to build an RTOS from scratch on STM32F407 Discovery.

## Current Features

- Cooperative scheduler
- Task states
- Task priorities
- Task delays
- Per-task stacks
- PSP/MSP exploration

## Hardware

STM32F407 Discovery

## IDE

STM32CubeIDE
